Bienvenida

Gracias por participar. La actividad tomará cerca de 20 minutos.
Este ejercicio es una forma diferente de aprender sobre la economía de los recursos naturales.
{% if planificador %} Usted es la única persona en este grupo (el grupo tiene 1 persona en total).
La idea es recrear una situación en que un pescador toma decisiones sobre el uso de los recursos de una pesquería.
{% elif planificador == False %} Usted ha sido asignado a un grupo con otras 3 personas (el grupo tiene 4 personas en total).
La idea es recrear una situación en que varios pescadores toman decisiones sobre el uso de los recursos de una pesquería.
{% endif %} Es muy importante que no converse con otras personas durante el ejercicio.

Instrucciones de pesca

Usted participará por varios periodos de tiempo equivalentes a semanas de trabajo.
Cada semana usted puede realizar {% if planificador %} entre cuatro y doce {% elif planificador == False %} entre uno y tres {% endif %} viajes de pesca dentro de un LAGO.

{% if deterministico %}

Cada viaje de pesca que usted realice le dará 3 UNIDADES DE PESCADO.

{% else %}

Cada viaje de pesca que usted realice puede ser BUENO, NORMAL o MALO.
Un viaje BUENO de pesca le dará 5 UNIDADES DE PESCADO.
Un viaje NORMAL de pesca le dará 3 UNIDADES DE PESCADO.
Un viaje MALO de pesca le dará 1 UNIDAD DE PESCADO.
Un viaje BUENO, NORMAL y MALO, tienen la misma probabilidad de ocurrir.

{% endif %} {% if planificador %} {% elif planificador == False %}

Ningún otro miembro de su grupo sabrá cuántos viajes de pesca realizó usted cada semana.

{% endif %} {% if deterministico == False %}

Los miembros de su grupo tampoco sabrán si cada viaje de pesca suyo fue BUENO, NORMAL o MALO.

{% endif %}

Recurso restante

Ejemplo:

En la primera semana la pesca {% if planificador %} {% elif planificador == False %} total entre los cuatro jugadores {% endif %} es de 20 UNIDADES DE PESCADO.
Esto significa que el lago tendrá 80 peces antes de comenzar la siguiente semana.
De acuerdo al diagrama, si hay 80 peces se reproducirán 5 peces más.
Para la semana 2 habrá 85 peces.

Ganancias

Cada UNIDAD DE PESCADO capturada durante el juego será pagada en puntos.

{% if precio_fijo %}

El precio pagado será de {{ precio_fijo_valor }} por unidad de pescado.

{% else %}

El precio en puntos depende de la pesca total del grupo cada semana.

Esta es la tabla de precios:

Extracción total del grupo 4-12 13-20 21-28 29-40 41-60
Precio por unidad 5 puntos 4 puntos 3 puntos 2 puntos 1 puntos
{% endif %}

Terminación del juego

Hay tres reglas de terminación:

  1. Si {% if planificador %} se {% elif planificador == False %} el grupo {% endif %} llega a la semana 10 y el lago aún tiene más de 12 peces.
  2. Si al comienzo de cualquier semana el lago tiene menos de 12 peces.
  3. Si la pesca total de una semana excede el número de peces que quedan en el lago.
{% if planificador %}

En este último caso, se extraen los peces restantes en el lago.

{% elif planificador == False %}

En este último caso, cada pescador recibe una cantidad de los peces restantes proporcional a la que habría sido de la pesca total.

{% endif %}

Por ejemplo, suponga que la pesca total del grupo fue 24 UNIDADES DE PESCADO. Sin embargo, en el lago solo quedan 20 peces.

El juego se terminará y usted recibirá 20 UNIDADES DE PESCADO.

{% if precio_fijo %}

Por otro lado, cada unidad será pagada a 2 puntos, como se mencionó anteriormente.
{% else %}

El precio, sin embargo, se determinará con la tabla de precios de acuerdo a la pesca total.
Es decir, en este caso cada unidad será pagada a 3 puntos.

{% endif %}
