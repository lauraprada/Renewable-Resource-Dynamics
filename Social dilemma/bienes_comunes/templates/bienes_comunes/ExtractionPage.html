Resumen de instrucciones

Cada semana usted puede realizar entre uno y tres viajes de pesca dentro de un LAGO.

{% if deterministico %}

Cada viaje de pesca captura {{ unidades_deterministicas }} UNIDADES de pescado.

{% else %}

Cada viaje de pesca captura en un día BUENO {{ unidades_dia_bueno }} UNIDADES de pescado, NORMAL {{ unidades_dia_regular }} UNIDADES de pescado, y en uno MALO {{ unidades_dia_malo }} UNIDAD de pescado.

{% endif %}

Ningún otro miembro de su grupo sabrá cuántos viajes de pesca realizó usted cada semana.

{% if deterministico %} {% else %}

Los miembros de su grupo tampoco sabrán si cada viaje de pesca suyo fue BUENO, NORMAL o MALO.

{% endif %}

La pesca total del grupo se sutraerá de los peces restantes en el lago.

Los peces restantes al inicio de cada semana se REPRODUCIRÁN de acuerdo al diagrama de arriba antes de comenzar la siguiente semana.

Cada UNIDAD de pescado se pagará en puntos.

{% if precio_fijo %}

El precio en puntos por unidad de pescado es de {{ precio_fijo_valor }}.

{% else %}

El precio por unidad de pescado depende del total de pesca en el grupo, según la siguiente tabla:

Extracción total del grupo 4-12 13-20 21-28 29-40 41-60
Precio por unidad 5 puntos 4 puntos 3 puntos 2 puntos 1 puntos
{% endif %}

El juego termina si:

  • El grupo llega a la semana {{ ultima_ronda }} y el lago aún tiene más de 12 peces.
  • Al comienzo de cualquier semana el lago tiene menos de 12 peces.
  • La pesca total de una semana excede el número de peces que quedan en el lago.
